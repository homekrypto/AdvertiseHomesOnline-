## 3. Subscription Management Workflows

### üîÑ **SUBSCRIPTION LIFECYCLE MANAGEMENT**

**New Subscription Flow:**
1. User selects plan ‚Üí Stripe checkout
2. Payment success ‚Üí Webhook updates user status
3. Welcome email sent ‚Üí Access granted immediately
4. Trial period tracking ‚Üí Automatic conversion

**Payment Failure Handling:**
1. Payment fails ‚Üí User flagged in dashboard
2. Grace period (3 days) ‚Üí Limited access maintained
3. Retry payment ‚Üí Automatic retry 3 times
4. Final failure ‚Üí Downgrade to free tier

**Cancellation Management:**
1. User cancels ‚Üí Subscription marked as "cancelled"
2. Access continues until period end
3. Auto-downgrade on expiration
4. Win-back email campaign triggered
